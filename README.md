#  Heart Disease Data Analysis

##  Project Overview
This project focuses on analyzing heart disease dataset to identify patterns and key risk factors influencing heart disease.

## 📊 Dataset
- Clinical dataset containing patient information such as age, sex, cholesterol, and heart disease status.
- Total records: ~1000+

## 🔍 Key Analysis Performed
- Disease distribution analysis (target variable)
- Age group vs heart disease analysis
- Gender-based disease comparison
- Cholesterol trends across age groups
- Cholesterol vs disease relationship

## 🛠 Tools Used
- Excel (Data cleaning & Pivot Tables)
- Basic Statistical Analysis

## 📈 Key Insights
- Middle-aged individuals show higher disease count
- Cholesterol levels vary across age groups
- Slight variation observed in cholesterol between diseased and non-diseased patients

## 📂 Project Structure
- `data/` → Raw dataset
- `analysis/` → Processed analysis file
