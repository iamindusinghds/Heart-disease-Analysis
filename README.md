# Heart Disease Risk Analysis using Data Analytics

## Project Overview
This project analyzes clinical data to identify patterns associated with heart disease.

## Tools Used
- Excel
- Data Analysis
- Pivot Tables

## Key Insights
- Higher disease cases observed in dataset
- Age shows increasing cholesterol trend
- Gender-based variation observed
- Cholesterol alone is not a strong predictor

## Conclusion
Heart disease is influenced by multiple factors and requires multi-variable analysis.
